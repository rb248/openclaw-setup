# Agent prompt templates

This directory contains **sanitized prompt files** for the OpenClaw agents I use
locally. They are intentionally kept free of private tokens, sessions, uploaded
files, and machine-specific credentials so the repository can stay public.

## Included agents

- `cv-writer`
- `job-search`
- `linkedin-content`
- `interview-prep`

## How to use

1. Copy the prompt templates into your local OpenClaw agent folders.
2. Replace any placeholders with your own private configuration.
3. Keep auth profiles, session logs, memory files, and uploaded documents out of GitHub.

## Safety rule

If a file contains tokens, personal contact details, session history, or uploaded
content, it belongs in your private `.openclaw/` directory — not in this repo.
