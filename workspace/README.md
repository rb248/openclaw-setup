# Workspace markdown templates

This folder mirrors the structure of the local OpenClaw workspace in a
**public-safe** way.

## Included files

- `AGENTS.md`
- `BOOTSTRAP.md`
- `HEARTBEAT.md`
- `IDENTITY.md`
- `MEMORY.md`
- `SOUL.md`
- `TOOLS.md`
- `USER.md`

## Important

The real local workspace contains private notes, identity details, and daily
memory files. Those should stay on the local machine and out of GitHub.

## Memory folder

The `workspace/memory/` folder in this repository is only for public-safe
explanations or templates. Do not publish real daily memory files there.
