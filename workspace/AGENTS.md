# Workspace - AGENTS

This is the public-safe template for a local OpenClaw workspace.

## Purpose
Provide startup guidance for the workspace without exposing private context.

## Startup order
1. Read `SOUL.md`
2. Read `USER.md`
3. Read recent `workspace/memory/` notes if they exist
4. Read `MEMORY.md` only in a private main session

## Safety
- Do not commit private memory, session history, or secrets
- Keep this file generic and reusable
